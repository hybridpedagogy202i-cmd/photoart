# מבט סטודיו צילום – אתר תדמית

אתר תדמית מקצועי לצלם אירועים המתמחה בצילום כנסים, תערוכות ואירועי חברה. האתר בנוי כ-**Static Site** עם דפי HTML נפרדים, CSS ייעודי ועיצוב רספונסיבי בעברית (RTL).

## 📄 מבנה האתר

| קובץ | תיאור |
|------|--------|
| `index.html` | דף הבית – הצגת שירותים, נתונים, תהליך עבודה ותיק עבודות נבחר |
| `portfolio.html` | גלריית פרויקטים מלאה |
| `portfolio-detail.html` | תבנית לדף פרויקט בודד עם גלריה מורחבת |
| `services.html` | חבילות צילום ומחירים |
| `about.html` | אודות הצלם/סטודיו ולוגואי לקוחות |
| `faq.html` | שאלות נפוצות |
| `contact.html` | טופס יצירת קשר ובדיקת זמינות |
| `privacy.html` | מדיניות פרטיות |
| `terms.html` | תנאי שימוש |

## 🖼️ תמונות ומדיה

כל התמונות ממוקמות בתיקייה `assets/images/`. קובץ [`IMAGES-LIST.md`](./IMAGES-LIST.md) מכיל רשימה מפורטת של כל התמונות הנדרשות, כולל:

- שמות קבצים מומלצים
- מידות מומלצות לכל תמונה
- טקסטי Alt ל-SEO
- המלצות לבחירה ועריכה

### להשקה ראשונית יש להכין:
- **~28 תמונות** לפחות (כולל תמונת Hero, קטגוריות, גלריה, לוגואי לקוחות)
- קובץ `favicon.ico` לאייקון האתר
- קובץ `og-image.jpg` לשיתוף ברשתות חברתיות

## 🎨 עיצוב וסגנון

- **פונט**: Heebo (Google Fonts)
- **CSS**: קובץ `assets/style.css` ייעודי עם משתני CSS לצבעים וטיפוגרפיה
- **RTL**: האתר מעוצב מימין לשמאל לתמיכה מלאה בעברית
- **רספונסיבי**: תומך במכשירי מובייל, טאבלט ודסקטופ

## 🚀 העלאה לאוויר

1. הכינו את כל התמונות בהתאם לרשימה ב-`IMAGES-LIST.md`
2. הניחו אותן בתיקייה `assets/images/`
3. העלו את כל קבצי האתר לשרת אינטרנט או אחסון סטטי (Netlify, Vercel, GitHub Pages וכו')
4. ודאו שהדפדפן מציג את התוכן בעברית כראוי

## 📝 התאמה אישית

- החליפו את טקסטי Lorem Ipsum בתוכן אמיתי
- עדכנו פרטי יצירת קשר בטופס ובפוטר
- הוסיפו/הסירו פרויקטים מתיק העבודות לפי הצורך
- התאימו את חבילות השירות והמחירים ב-`services.html`
